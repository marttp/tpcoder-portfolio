ผมสมัคร Medium Premium ตั้งแต่ปีที่เรียนจบวิศวกรรมคอมพิวเตอร์เลยครับ ตอนนั้นรู้สึกว่ามันเหมือนค่าเรียนรายเดือนมากกว่าของฟุ่มเฟือย หลัง paywall มีบทความ system design เชิงลึก มีประสบการณ์จริงจากรุ่นพี่ที่ผ่านงานหนักมา เดือนละห้าดอลลาร์เพื่อเรียนรู้จากคนเหล่านี้ ไม่ต้องคิดมากครับ กดสมัครเลย

แต่หลายปีให้หลัง ผมเริ่มตั้งคำถามที่ไม่เคยนึกว่าจะถาม: **มันยังคุ้มอยู่ไหมวะ?**

โพสต์นี้ไม่ได้จะมาด่านะครับ แต่เป็นการทบทวนจริงจังจากคนที่เคยรักแพลตฟอร์มนี้ แล้วค่อยๆ รู้สึกว่าสิ่งที่ได้กลับมามันไม่คุ้มเหมือนเดิม

---

## ยุคทองนั้นมีจริง

พูดให้แฟร์ก่อนครับ ช่วงแรกๆ ผมได้ของดีจริง เจอบทความเรื่อง caching strategies, distributed system pitfalls, trade-offs ของ event-driven architectures — เขียนโดยคนที่ลงมือทำจริง มีรอยแผลจากการ deploy ตอนตีสามมาเล่าให้ฟัง ไม่ใช่ tutorial เอามาเขียนใหม่ มันคือประสบการณ์ที่กลั่นออกมาเป็นตัวหนังสือ

ผมยังจำได้ว่าเคยเจอบทความเรื่อง rate-limiting patterns ที่เอาไปใช้กับระบบจริงได้เลย อีกบทความเรื่อง consistent hashing ที่ docs ทางการไม่เคยอธิบายได้ชัดขนาดนั้น

แต่ยุคนั้นจบไปแล้วครับ

---

## อะไรเปลี่ยนไป

### 1. AI ปิดช่องว่างความรู้หมดแล้ว

ทุกวันนี้ LLM ตอบได้เกือบหมดในสิ่งที่บทความ Medium เคยให้ อยากเข้าใจ consistent hashing? เทียบ event sourcing กับ CQRS? ถาม AI ได้เลย ตอบตามบริบทเรา โต้ตอบได้ทันที แถมฟรี

บทความแนว "มาทำความเข้าใจ concept X กัน" ตอนนี้กลายเป็นความรู้ที่หาได้ทั่วไป AI ไม่ได้แค่ทำได้เท่า แต่หลายครั้งดีกว่าด้วยซ้ำ เพราะปรับตามสิ่งที่เราถามได้และถามต่อได้เลย

### 2. เนื้อหาคุณภาพตกลงเรื่อยๆ

เปิด Medium ตอนนี้จะเจอหัวข้อเดิมวนซ้ำ "10 สิ่งที่ developer ต้องรู้" "ทำไม microservices อาจไม่เหมาะกับคุณ" — เขียนโดยคนที่อ่านหนังสือสามเล่มเดียวกับเรา แล้วแพ็คใหม่เป็นโพสต์ห้านาทีเพื่อเอารายได้จาก Partner Program

บางบทความแค่เอา docs มาเขียนใหม่ ผมเคยคลิกเข้าบทความ premium แล้วพบว่าอ่านได้ฟรีบน docs ของ framework นั้นเลย docs ต้นฉบับถูกต้องกว่า อัปเดตกว่า แถมฟรี

บทความจากประสบการณ์จริง มีมุมมองที่คิดมาดี มีเรื่องเล่าจากปัญหาแปลกๆ ในระบบจริง — หายากขึ้นเรื่อยๆ ครับ

### 3. ระบบจูงใจมันพัง

นี่คือต้นตอเลยครับ Partner Program จ่ายตามยอด engagement, engagement สูงเมื่อหัวข้อจั่วๆ ดึงคลิกได้เยอะ คนก็แข่งกันเขียนเยอะแต่ไม่ลึก เนื้อหาตื้นๆ ท่วม feed หมด

นักเขียน premium จำนวนไม่น้อยไม่ได้เขียนเพราะอยากแบ่งปันความรู้ แต่เขียนเพื่อหาเงินจากคลิก สำหรับคนที่ต้องสร้างระบบจริงๆ การมานั่งคัดแยกหาของดีสักชิ้น มันเหนื่อยเกินไป

แล้วคนที่เก่งจริง? หลายคนย้ายไป blog ส่วนตัว, Substack หรือ engineering blog ของบริษัทไปหมดแล้วครับ

## เมื่อไหร่ที่ยังพอคุ้ม?

ไม่ได้บอกว่ามันไม่มีค่าเลยนะครับ ยังมีบางกรณีที่โอเค:

**เพิ่งเริ่มทำงาน** — น้องๆ junior ที่ยังสร้างฐานอยู่ บทความที่คนเขียนจริงมีเรื่องเล่าและลำดับการอธิบายที่ AI ไม่ได้ให้เสมอไป ผมเองก็เคยได้ประโยชน์ตรงนี้ตอนปีแรกๆ

**ติดตามนักเขียนเฉพาะคน** — ยังมีคนหยิบมือที่เขียนของดีอยู่ เอาประสบการณ์จากระบบจริงมาเล่า มีข้อมูลสนับสนุน ถ้าตามอ่านคนเหล่านี้อยู่ ค่าสมาชิกก็อาจคุ้ม แต่คุณจ่ายให้คนสามสี่คน ไม่ใช่แพลตฟอร์ม

**ชอบเจอของที่ไม่ได้ตั้งใจหา** — AI ตอบได้ดีถ้าเรารู้ว่าจะถามอะไร แต่บทความคนเขียนบางทีทำให้เจอมุมมองที่ไม่เคยนึกถึง แต่มันก็หายากขึ้นเรื่อยๆ ท่ามกลางเนื้อหาที่ไม่ได้เรื่อง

---

## ลองคิดเลขดู

Medium Premium ปีละ $60 ครับ ใน 12 เดือนที่ผ่านมา มีกี่บทความ premium ที่ให้อะไรจริงๆ ที่หาไม่ได้จาก docs, AI หรือ engineering blog ฟรี?

ถ้าน้อยกว่าห้า แปลว่าจ่าย $12 ต่อบทความที่มีประโยชน์จริงสักชิ้น มันไม่คุ้มครับ เอาเงินไปซื้อหนังสือดีๆ สักเล่มอ่านให้มันลึกๆ ยังคุ้มกว่า

---

## ตอนนี้ผมเรียนรู้จากไหน

**Documentation ทางการ** — ข้อมูลจากต้นทาง ฟรี มักเขียนดีกว่าบทความที่สรุปมาอีกที

**Engineering blog บริษัทเทค** — Uber, Stripe, Netflix, Cloudflare เอาเรื่องจากระบบจริงมาเล่า มี data สนับสนุน ไม่มี paywall

**AI** — สำรวจ concept ใหม่ ทดสอบความเข้าใจ ถามตอบเร็วๆ ไม่ perfect แต่เร็วและโต้ตอบได้ดี

**หนังสือที่เลือกมาตรงจุด** — อย่าง _System Design Interview_ หรือ _The Software Engineer's Guidebook_ ให้คุณค่าระยะยาวมากกว่าบทความ Medium หลายร้อยชิ้นรวมกัน

**อ่าน open-source code** — อ่าน code จริงสอนได้มากกว่าอ่านบทความเกี่ยวกับ code

---

## สรุปตรงๆ

Medium Premium เคยคุ้มมากตอนที่คนเก่งๆ ยังเขียนอยู่ที่นี่ วันนี้คนเก่งย้ายไปที่อื่น AI สอนพื้นฐานแทนได้ แล้วระบบของแพลตฟอร์มก็ดันเนื้อหาคุณภาพต่ำขึ้นมาเสมอ

ถ้ายังได้ประโยชน์จริงอยู่ ก็เก็บไว้ครับ แต่ถ้าต่ออายุเพราะเคยชินเหมือนผม ลองย้อนดูว่าปีที่ผ่านมาอ่านอะไรจริงๆ บ้าง

สิ่งที่ผมทำตอนนี้? ลองยกเลิกดูก่อนครับ แล้วดูว่ามันกระทบชีวิตประจำวันจริงไหม ถ้าไม่ — ซึ่งผมค่อนข้างมั่นใจว่าไม่ — ก็ถึงเวลาบอกลาจริงๆ แล้วล่ะ ยังไงก็ตาม $60 นั้นเอาไปซื้อหนังสือดีๆ สักเล่มหรือจดโดเมนตั้งบล็อกของตัวเองดีกว่า เอาเวลาไปสร้างมากกว่าบริโภคครับ

_ทั้งหมดนี้เป็นความเห็นส่วนตัวของผม จากประสบการณ์หลายปีที่ค่อยๆ รู้สึกว่ามันไม่คุ้มเหมือนเดิม แต่จากที่คุยกับ engineer คนอื่นๆ มา ผมคิดว่าหลายคนน่าจะรู้สึกคล้ายๆ กันครับ_
